# 1、项目说明

## 1.1、项目初衷

利用空闲时间，通过完成心跳监测工具，实践Netty技术，加深对Netty技术理解与应用

## 1.2、项目目标

+ 封装心跳监测starter，用于服务启动状态监测
+ 后端管理系统，进行监测规则配置，服务状态可视化管理

## 1.3、技术栈

+ Netty
+ Vue
+ Spring boot
+ ……
